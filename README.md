# Projeto de Firewall

Este projeto é um gerenciador de firewall para o `iptables` do Linux, utilizando o `dialog` para uma interface de usuário amigável.

## Instalação

Para instalar as dependências, execute o script `install.sh`:
```bash
bash install.sh
```

## Uso

Para executar o gerenciador de firewall, execute o script `firewall.sh`:
```bash
bash firewall.sh
```

Para informações mais detalhadas, consulte o arquivo `MANUAL_FIREWALL.md`.
